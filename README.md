# CSCI 5607: Project 4
## Real-Time 3D Game Programming

### To compile on Mac OS: 
```g++ *.cpp -x c glad/glad.c -g -F/Library/Frameworks -framework SDL3 -framework OpenGL -Wl,-rpath,/Library/Frameworks -o game```

### Keyboard/Mouse Control:
1. Arrow keys to move forward/backward/left/right (supports strafing/sidestepping)
2. Move the mouse to control the first-person camera
